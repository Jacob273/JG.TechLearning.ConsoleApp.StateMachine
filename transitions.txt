[*] --> Idle : Initialize

Idle --> Initialized : Initialize
Initialized --> ElementPlacement: Start
ElementPlacement--> Welding : Run
Welding --> Idle: Complete

Idle -> Error : Failed
InitializedState --> Error : Failed
ElementPlacement--> Error : Failed
Welding --> Error : Failed
